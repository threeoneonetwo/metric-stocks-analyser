# Metric Finance Design Tokens

These tokens lock the MVP visual direction.

| Token | Value | Usage |
| --- | --- | --- |
| Yellow canvas | `#FFEA3D` | Page background and brand field |
| Ink | `#0A0A0A` | Primary text |
| Border black | `#000000` | One pixel borders and footer bar |
| Cream surface | `#FAF7F0` | Cards, sections, modal surfaces |
| Bullish green | `#00C46A` | Positive deltas and live dots |
| Bearish red | `#E5484D` | Negative deltas and risk emphasis |
| Link blue | `#1F4DD6` | Rare links and selection accents |

Typography:

| Role | Font |
| --- | --- |
| Headlines | Source Serif 4, 700 |
| Body and UI | Geist, 400 to 500 |
| Labels and numeric chrome | JetBrains Mono |
